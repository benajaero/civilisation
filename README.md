# civilisation

Launch-foundation bootstrap for a monorepo built around governance-first publishing.

- Monorepo bootstrap establishes the workspace layout and shared TypeScript baseline.
- Governance and provenance rules apply before publication and across all consumer apps.
- `packages/brand` is the canonical source for brand and editorial content.
- `apps/web` is the consumer shell that proves the homepage and deployment path.
- Content moves through review, provenance, and contribution controls before it is published.
