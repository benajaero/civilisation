# Content Provenance

All published content must have a clear source, owner, and review trail before it is merged or shipped.

## Source Adapter Rights Checks

Source adapters may normalize upstream metadata, links, and candidate text, but they do not make content publishable on their own. Every imported work still needs editorial provenance review before publication.

Project Gutenberg candidates must be checked against the license text inside the ebook. Works marked as not restricted by U.S. copyright law can proceed to editorial review only after the reviewer confirms the license status and either preserves the required Project Gutenberg license/trademark terms or removes Project Gutenberg trademark/license matter from the republished text. Copyrighted works distributed to Project Gutenberg by permission require separate permission from the copyright holder before republication.

Chinese Text Project candidates must use the JSON API for limited, reasonable textual access and must not scrape site pages. CTP translations and site content require rights review, and translations must not be copied without permission from the relevant copyright holder. Citations should link back to the corresponding CTP page.

Reviewed source terms:

- Project Gutenberg license: `https://www.gutenberg.org/policy/license`
- Project Gutenberg robot access: `https://www.gutenberg.org/policy/robot_access.html`
- Chinese Text Project API terms: `https://ctext.org/tools/api`
- Chinese Text Project FAQ/footer terms: `https://ctext.org/faq`
